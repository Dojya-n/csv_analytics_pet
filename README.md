# CSV Analytics

Консольный анализатор данных продаж на Python.

## Возможности
- Подсчёт общей выручки
- Аналитика продаж по категориям
- Топ N самых продаваемых товаров
- Продажи по месяцам
- Сохранение отчёта в report.txt
- Обработка ошибок: отсутствующий файл, неверный формат, пустые поля

## Структура CSV файла
date, product, category, amount, quantity
2024-01-15, Ноутбук, Электроника, 50000, 1
2024-01-16, Мышь, Электроника, 1500, 3

## Запуск
```bash
python csv_analytics.py
```

## Результат
Программа выводит аналитику в консоль и сохраняет отчёт в `report.txt`

## Технологии
- Python 3.11
- csv
- collections
- datetime
